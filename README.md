# 📘 English Tuition Interactive Website

An interactive front-end website developed for an English tuition institute.  
The project is built using HTML, CSS, and JavaScript and provides structured learning resources, course details, and exam materials in a user-friendly interface.

## 🚀 Features

- 🏠 Home / Landing Page
- ℹ️ About Us Page
- 📞 Contact Us Page
- 📚 Courses and Course List Pages
- 📝 Exam Papers Section
- 🎨 Interactive UI with JavaScript functionality
- 🖼️ Organized resources (Images, PDFs, Posters)

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)
